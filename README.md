Задание
Реализовать умножение двух квадратных матриц размером от 100×100 до 2000×2000 с проверкой корректности и замером времени. Реализация включает две функции: на CPU и на GPU с использованием CUDA.

Язык: Python
Среда: Google Colab

Описание реализации
CPU (mult_matr_cpu)
Используется блочный алгоритм (tiling) с размером блока 32×32. Матрица разбивается на блоки, что улучшает использование кэш-памяти процессора. Сложность алгоритма — O(n³).

GPU (mult_matr_gpu)
Матрицы копируются из оперативной памяти в память видеокарты

Настраивается сетка потоков (блоки 16×16)

Выполняется «прогрев» для исключения времени JIT-компиляции

Каждый поток GPU вычисляет один элемент результирующей матрицы

Ядро CUDA (mat_mul)
Каждый поток получает уникальные индексы i и j, после чего вычисляет скалярное произведение i-й строки матрицы A и j-го столбца матрицы B.

Что и почему распараллелено?
Сложность задачи O(n³) приводит к быстрому росту времени выполнения на CPU. При этом вычисления каждого элемента результирующей матрицы независимы, что позволяет выполнять их параллельно.

Распараллелены два внешних цикла (по строкам i и столбцам j). В GPU-версии эти циклы полностью отсутствуют — вместо последовательного обхода, видеокарта вычисляет все ячейки одновременно.
